wxstream

wxstream is a tool made by nicbudd to monitor iembot messages in a minimalistic, highly configurable frontend. It's a personal project by nicbudd intended to replace weather.im for monitoring weather events, particularly while storm chasing.

Usage

Click the button to add a new feed and the button to remove it

Click on the timestamp of the message to jump to the IEM VTEC browser where you can see the full information

Feeds

Currently the website is configured with the options for feeds:

  • Default: reflects the "botstalk" channel with several types of messages filtered out, including METAR, TAF, ZFP, AFD, PIREP.
  • Severe: default feed with only talk about severe weather and tornadoes.
  • Tornado: default feed with only talk about tornadoes.
  • Winter: default feed with only winter, snow, and wintry mix information.
  • Fire: default feed with only fire information.
  • METAR: default feed with only fire information.
  • Damage: default feed with only storm damage information.
  • All:the pure "botstalk" without any filtering.
  • BOX:NWS Boston feed, with the same whitelist as Default.
  • GYX:NWS Gray feed, with the same whitelist as Default.

This is still actively being developed, expect changes and new features, including the ability to modify feeds.
